# Hunger Games Analytics Simulator

## Aperçu
Projet de simulation de type "Battle Royale" inspiré d'une tendance Instagram.  
Le script génère une population réaliste, simule des combats anatomiques tour par tour et analyse la survie d'un joueur personnalisé.  

Conçu pour vérifier les calculs simplistes des influenceurs et fournir une estimation statistique précise jusqu'à quel round un individu peut survivre.

## Objectifs
- Fournir une simulation réaliste basée sur la biomécanique (poids, taille, âge) et la démographie mondiale.
- Simuler l'état mental et physique des combattants (blessures aux membres, stress post-traumatique).
- Permettre l'intégration d'un "Joueur" avec ses propres stats pour prédire son destin dans le tournoi.
- Offrir une visualisation graphique de l'évolution de la population (pyramide des âges, santé, richesse).

## Technologie
- Python (Google Colab)

## Déploiement
Voici les étapes rapides pour lancer la simulation et connaître votre destin :

- Ouvrez le notebook dans Google Colab.
- Exécutez les cellules d'initialisation (Imports, Classe Human, Fonctions).
- Modifiez la fonction `integrer_joueur` dans la section **"Tournoi avec moi"** avec votre taille, poids, âge et niveau d'entraînement.
- Lancez la cellule principale `hunger_games` → patientez durant la simulation des combats.
- À la fin, consultez le journal d'exécution pour voir à quel round vous avez été éliminé et le profil du champion.
